## What is AI?

### 定义
- 创造能执行类人任务的智能机器
- 五大能力：Learning, Reasoning, Problem Solving, Perception, Language

### Turing Test (1950)
- 人类无法区分 AI 与人类 → AI 通过测试
- Alan Turing 提出
- [[Week01/notes#The Turing Test (1950)|详细说明]]

## AI 历史

### 三次浪潮
- 1950s-60s: 第一次 AI 热潮（Turing Test）
- 1980s: 第二次 AI 热潮（Expert Systems）
- 2010s-now: 第三次 AI 热潮（Deep Learning, LLMs）

### 关键时间点
- 1956: Dartmouth Conference（AI 诞生）
- 2012: Deep Learning 突破
- 2022-23: GPT-4, ChatGPT

## Expert Systems (1980s)

### 架构
- Knowledge Base（知识库）
- Inference Engine（推理引擎）
- Explanation System（解释系统）

### 特点
- 基于 IF-THEN 规则
- 没有学习能力
- 例子：MYCIN 医疗诊断

### 现代应用
- 客服系统
- 金融顾问
- 工程诊断

## Fuzzy Logic（模糊逻辑）

### 概念
- 扩展经典 True/False 到 0-1 之间
- Membership degree（隶属度）

### 应用
- 电饭煲、ABS 刹车
- 风扇速度控制

## Neural Networks（神经网络）

### 结构
- Input Layer → Hidden Layer(s) → Output Layer
- Fully Connected / Dense Networks
- Deep = 多个隐藏层

### 硬件需求
- GPU（NVIDIA 主导）
- PyTorch, TensorFlow, Keras

### 关键人物
- Geoffrey Hinton: "Godfather of AI"
- Yann LeCun: CNN 发明者

## Machine Learning 基础

### 定义
- 从数据中找模式（patterns）
- 对未见数据做预测

### ML Pipeline
- 定义问题 → 收集数据 → 预处理 → 选模型 → 训练 → 评估 → 部署

### 三大类型
- Supervised Learning: 有标签
- Unsupervised Learning: 无标签
- Reinforcement Learning: 奖惩反馈

### Classification vs Regression
- Classification: 离散类别（Cat vs Dog）
- Regression: 连续数值（房价）
