# CSCI 6751 - Lecture 01: Vocabulary & Key Terms

## Core AI Concepts

| Term | Definition |
|------|------------|
| **Artificial Intelligence (AI)** | Field of computer science focused on creating machines capable of intelligent behavior - learning, reasoning, and problem-solving |
| **Machine Learning (ML)** | Subset of AI where systems learn patterns from data without being explicitly programmed |
| **Deep Learning** | Subset of ML using neural networks with multiple hidden layers |
| **Neural Network** | Computing system inspired by biological neurons, consisting of interconnected nodes that process information |
| **Turing Test** | Test proposed by Alan Turing (1950) to determine if a machine exhibits intelligent behavior indistinguishable from a human |

## Machine Learning Types

| Term | Definition |
|------|------------|
| **Supervised Learning** | ML approach using labeled training data (inputs with known outputs) |
| **Unsupervised Learning** | ML approach using unlabeled data to find hidden patterns or groupings |
| **Reinforcement Learning** | ML approach where an agent learns through trial and error, receiving rewards or penalties |
| **Classification** | Predicting discrete categorical labels (e.g., spam/not spam, cat/dog) |
| **Regression** | Predicting continuous numerical values (e.g., house prices, temperature) |
| **Clustering** | Grouping similar data points together without predefined labels |

## Neural Network Terms

| Term | Definition |
|------|------------|
| **Neuron** | Basic unit in a neural network that receives inputs, applies weights, and produces an output |
| **Input Layer** | First layer of a neural network that receives raw data |
| **Hidden Layer** | Intermediate layers between input and output that perform computations |
| **Output Layer** | Final layer that produces the network's prediction |
| **Fully Connected (Dense) Layer** | Layer where every neuron connects to all neurons in the adjacent layer |
| **Deep Neural Network** | Neural network with many hidden layers (potentially thousands) |
| **GPU (Graphics Processing Unit)** | Hardware accelerator essential for training large neural networks |

## Expert Systems

| Term | Definition |
|------|------------|
| **Expert System** | AI program that emulates decision-making of a human expert using if-then rules |
| **Knowledge Base** | Database of facts and rules derived from human experts |
| **Inference Engine** | Component that applies logical rules to the knowledge base to derive conclusions |
| **MYCIN** | Famous early expert system for medical diagnosis (1970s) |

## Fuzzy Logic

| Term | Definition |
|------|------------|
| **Fuzzy Logic** | Extension of classical logic allowing degrees of truth between 0 and 1 |
| **Membership Degree** | Value (0-1) indicating how much an element belongs to a fuzzy set |
| **Crisp Logic** | Traditional binary logic with only true (1) or false (0) values |

## Data & Training

| Term | Definition |
|------|------------|
| **Training Data** | Dataset used to teach a model to recognize patterns |
| **Testing Data** | Unseen data used to evaluate model performance |
| **Labeled Data** | Data where each sample has a known target/outcome |
| **Unlabeled Data** | Data without known targets, used in unsupervised learning |
| **Data Augmentation** | Technique to increase training data by applying transformations (rotation, flipping, etc.) |
| **Feature** | Individual measurable property of data used as input to a model |
| **Ground Truth** | The actual correct value/label for a data point |

## Model Evaluation

| Term | Definition |
|------|------------|
| **Loss/Error** | Difference between predicted and actual values |
| **Accuracy** | Proportion of correct predictions |
| **Training** | Process of adjusting model parameters to minimize error on training data |
| **Prediction/Inference** | Using a trained model to make predictions on new data |

## Linear Regression Terms

| Term | Definition |
|------|------------|
| **Linear Regression** | Model that fits a straight line to predict continuous values |
| **Non-linear** | Relationship that cannot be represented by a straight line |
| **Minimum Error** | Goal of regression: find the line that minimizes total prediction errors |

## Python & Tools

| Term | Definition |
|------|------------|
| **NumPy** | Python library for numerical computing with arrays |
| **PyTorch** | Deep learning framework developed by Facebook |
| **Keras** | High-level neural network API |
| **scikit-learn** | Python library for traditional machine learning algorithms |
| **Jupyter Notebook** | Interactive development environment for Python |
| **Google Colab** | Free cloud-based Jupyter notebook environment |
| **IDE** | Integrated Development Environment (PyCharm, VS Code) |

## Data Structures (Python)

| Term | Definition |
|------|------------|
| **List** | Ordered, mutable collection in Python using `[]` |
| **Array (NumPy)** | Efficient numerical array supporting vectorized operations |
| **Matrix** | 2D array (array of arrays) |
| **Slicing** | Extracting portions of a list/array using `[start:end]` notation |
| **RGB** | Color model using Red, Green, Blue values (0-255 each) |

## Historical Terms

| Term | Definition |
|------|------------|
| **Dartmouth Conference (1956)** | Workshop where "Artificial Intelligence" was first coined as a field |
| **AI Winter** | Period of reduced funding and interest in AI research |
| **AI Boom** | Period of rapid advancement and excitement in AI |
| **Transformer** | Neural network architecture behind modern LLMs (GPT, etc.) |
| **LLM (Large Language Model)** | AI models trained on massive text data (GPT-3, GPT-4, ChatGPT) |
