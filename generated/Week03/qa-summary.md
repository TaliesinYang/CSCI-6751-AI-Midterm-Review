# Week03 Q&A Summary — Linear Regression

> 从课堂互动中提取的问答/高频追问（按主题归类）。

---

## 1) 考试：能不能用 Python？
**Q**：能用 Python 来算/求解吗？考试怎么办？

**A（教授要点）**：
- 作业/练习中可以用 Python 辅助。
- 考试场景可能没有 Python，因此需要理解推导与计算步骤。
- 关键公式需要记住，尤其是线性回归闭式解。

---

## 2) 考试：只给数据要我们做什么？
**Q**：如果只给 X 和 Y 数据，我们怎么知道用几次（degree）/怎么建模？

**A（教授要点）**：
- 考试一般会给你一个假设（例如“假设是线性函数/一阶/二阶”）。
- 在给定假设下，你需要会用对应方法求解。

---

## 3) 闭式解：我们要记什么？
**Q**：闭式解需要记哪些东西？

**A（教授要点）**：
- 需要记住 **normal equation**：
  - **θ = (Xᵀ X)⁻¹ Xᵀ y**
- 需要会基本线性代数：转置、求逆、行列式（与求逆相关）。

---

## 4) 任务类型：回归还是分类？
**Q**：这是回归还是分类？

**A（教授要点）**：
- 看输出：
  - 连续数值（例如住院天数、长度）→ 回归
  - 离散类别（标签）→ 分类

---

## 课堂关键结论（教授反复强调）
1. Regression vs classification：看输出类型
2. Linear regression：闭式解 vs 梯度法
3. 考试要背 normal equation + 会求逆
